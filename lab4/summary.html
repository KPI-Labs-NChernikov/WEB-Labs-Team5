ВИСНОВКИ

В результаті виконання лабораторної роботи, ми придбали практичні навички роботи - маніпулювання інформаційним вмістом Web-документа засобами мови JavaScript, використання об’єктів, масивів, функцій, подій, обробників подій у сценаріях на мові JavaScript.

Бригадою було зроблено завдання 2 та розроблено JavaScript-сценарій для головної сторінки сайту кінотеатру, де головний постер змінюється кожні 20 секунд, а також можна його гортати вправо та вліво вручну (до завдання 8).

Також, кожен студент окремо виконав завдання 4 та 6: на використання масивів у мові JavaScript та на застосування технології DOM - об'єктної моделі документа.

JavaScript використовують для керування сценаріями перегляду гіпертекстових Web сторінок на стороні клієнта.
