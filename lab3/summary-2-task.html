Откаленко Орина:
18 за списком

Лабораторна робота була виконана в середовищі Visual Studio Code на мовах розмітки HTML та CSS. Результатом роботи стало знайомство з різними видами верстки сайтів (таблична, блочна) та отримання навичок роботи у реалізації різних макетів веб-сторінок.
Блочна верстка та плаваючі блоки є більш універсальними за табличний метод верстки через більшу кількість можливих варіантів оформлення інтерфейсу сайту з їх допомогою

Семенюк Даніїл:
26 за списком

Виконавши 2 завдання у лабораторній роботі, я виявив, що таблична верстка не є універсальною, її можна було б використовувати для невеличких сайтів, які не потребують багатьох блоків інформації. Зручніше використовувати плаваючі блоки, ми можемо робити з ними буквально все що завгодно, тому вони є універсальними, і через це є популярними серед людей які верстають сайти

Станков Артем:
28 за списком

Таблична верстка не є універсальною, її доцільно використовувати лише для сайтів, які не потребують багатьох блоків інформації. Зручніше використовувати плаваючі блоки, тому, що вони є більш універсальними, і через це значно більше використовуються у верстанні сайтів

Черніков Нікіта:
30 за списком

В результаті виконання 2 завдання лабораторної роботи №3, я навчився верстати веб-сторінки за допомогою таблиць та плаваючих блоків. Дійшов до висновку, що плаваючі блоки є значно кращими для розмітки сторінок.
По-перше, є значно більше можливостей для стилізації сторінки, зробленої за допомогою плаваючих блоків, ніж для зверстаної за допомогою таблиць.
По-друге, код виглядає значно більш зрозумілим, зі звичними назвами елементів, та простішим.
По-третє, плаваючі блоки є значно більш гнучкими щодо їх розміщення на веб-сторінці.
Отже, для розмітки веб-сторінки краще використовувати плаваючі блоки, аніж таблиці.
